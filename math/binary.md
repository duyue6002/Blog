# 二进制

## 原码、反码、补码

### 原码

> 8 位中的 6: 00000110
>
> 8 位中的-6: 10000110

### 反码

> 8 位中的 6: 00000110 与原码相同
>
> 8 位中的-6: 11111001 符号位不变，其余取反

### 补码

> 8 位中的 6: 00000110 与原码相同
>
> 8 位中的-6: 11111010 反码 +1

## 位运算

1. 左移，`<<`，翻倍，注意溢出
2. 右移
   1. `>>>`，逻辑右移，最左补 0，一般只用于正数
   2. `>>`，算术右移，符号位不变，负数在符号位后补 1，正数补 0
3. 或，`|`，有 1 则 1
4. 与，`&`，全 1 则 1
5. 异或，`^`，异则 1，同则 0
6. 非，`~`，取反

### 应用

#### 向下取整

```js
let a = (Math.random() * 10) | 0;
let b = Math.floor(Math.random() * 10);
```

#### 判断奇偶

```js
2 & 1; // 0
3 & 1; // 1
```
